# 🏠 ML House Price Prediction Dashboard

A full-stack Machine Learning web application built using Flask that allows users to predict house prices, analyze data, and generate professional reports.

---

## 🚀 Project Overview

This project is an end-to-end ML system where users can:

- 🔐 Create an account and login securely  
- 🤖 Predict house prices using trained ML models  
- 📊 Visualize analytics and model performance  
- 📁 Upload datasets and retrain models  
- 📄 Generate professional PDF reports with charts & insights  

---

## ✨ Features

- 🔐 User Authentication (Signup/Login)
- 🤖 Machine Learning Prediction
- 📊 Analytics Dashboard
- 📁 CSV Upload & Model Training
- 📈 Charts (Line, Bar, Pie)
- 📄 PDF Report Generation (with insights)
- 📥 Download CSV
- 📧 Email Report
- 🎨 Modern UI (Dark Theme)

---

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, Bootstrap  
- **Backend:** Flask (Python)  
- **ML:** Scikit-learn  
- **Database:** SQLite  
- **Charts:** Matplotlib  
- **PDF:** ReportLab

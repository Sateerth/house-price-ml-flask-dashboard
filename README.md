# ML House Price Dashboard

## Features
- 🔐 User Authentication (Email Login)
- 🤖 ML Prediction System
- 📊 Analytics Dashboard
- 📁 CSV Upload & Training
- 📄 PDF Report Generation
- 📈 Charts & Insights

## Tech Stack
- Flask
- Scikit-learn
- SQLite
- Bootstrap
- ReportLab

## Run Project
pip install -r requirements.txt  
python app.py
